| Shortcut | Action |
|----------|--------|
| `zc`     | Close the current fold |
| `zC`     | Close all folds recursively under the cursor |
| `zo`     | Open the current fold |
| `zO`     | Open all folds recursively under the cursor |
| `za`     | Toggle the current fold open/closed |
| `zA`     | Toggle all folds recursively under the cursor |
| `zM`     | Close **all** folds in the file |
| `zR`     | Open **all** folds in the file |
| `zm`     | Increase the fold level (show less) |
| `zr`     | Decrease the fold level (show more) |
| `[z`     | Jump to start of the current open fold |
| `]z`     | Jump to end of the current open fold |
